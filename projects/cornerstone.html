Description

This was the final project for my cornerstone 2 class (GE1502) and my first engineering project. For this project, we were tasked with making a project that implemented biomimicry to address a problem. We decided to make a contraption inspired by a Venus Flytrap to attract and squash flies. To do this, we used LED lights to attract flies and had a motion sensor that would trigger the motor to release the rubber band holding a platform in place. See video below for a demonstration.
